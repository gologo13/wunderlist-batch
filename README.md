# wunderlist-batch
Batch interface for Wunderlist

## Installation

```shell
$ npm install wunderlist-batch -g
```

## Usage

```js
$ wlbatch configure <accessToken> <clientId>

$ wlbatch create 

$ wlbatch destroy [batchId]

$ wlbatch list
```

Please make sure a daemon process is supposed to run at time when you expect.
